## üöÄ Features
- **Speech-to-Text:** Converts lecture audio to text using [Whisper/Other].
- **AI Summarization:** Uses **Ollama** to structure raw text into bullet points.
- **Image Recognition:** Extracts text from blackboard photos (OCR).
- **Local Privacy:** Runs entirely on your machine (Powered by RTX 5060).
